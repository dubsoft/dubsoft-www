Five Guys Mobile Apps

dubsoft teamed up with Solertium to help build the Five Guys Android & iOS Apps. We integrated with OLO's online ordering API to build an awesome user experience for Five Guy's Fans!

The Android application was built using the standard Java Android SDK and the iOS app was built using JavaScript and the Titanium framework.

Solertium created a wonderful in-app feedback and error reporting SaaS named Appygram. dubsoft created and maintains the 3rd party connectors for nodejs and Titanium. They are open source and available on github.

  • appygram-node
  • appygram-ti
